### 📄 Program Details
The MotorPH Payroll System is a Java-based application designed to automate salary calculations. It functions by reading employee information and attendance records from CSV files, calculating the total hours worked, and determining the net pay after applying necessary deductions like SSS, PhilHealth, and Tax.
